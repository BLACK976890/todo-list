# Todo List App

Учебный проект «Список дел» на React + TypeScript.  
Позволяет добавлять, удалять, отмечать выполнение и редактировать задачи, фильтровать их по статусу (все / активные / завершённые), а также переключать светлую и тёмную тему. Состояние сохраняется в localStorage браузера.

## 🛠️ Стек

- **React 18** (функциональные компоненты, хуки)
- **TypeScript**
- **Vite** (сборка)
- **CSS** (кастомная тёмная/светлая тема + базовые стили TodoMVC)
