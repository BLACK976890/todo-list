Todo List App
Учебный проект «Список дел» на React + TypeScript.
Позволяет добавлять, удалять, отмечать выполнение и редактировать задачи, фильтровать их по статусу (все / активные / завершённые), а также переключать светлую и тёмную тему. Состояние сохраняется в localStorage браузера.


Стек
React 18 (функциональные компоненты, хуки)
TypeScript
Vite (сборка)
CSS (кастомная тёмная/светлая тема + базовые стили TodoMVC)



Как запустить локально
1 Склонируйте репозиторий:
2 git clone https://github.com/BLACK976890/todo-list.git
3 Перейдите в папку проекта: cd todo-list
4 Установите зависимости: npm install
Запустите: npm run dev


Деплой
Проект развёрнут на Vercel: https://todo-list-flax-rho.vercel.app


Функциональность:
1 Добавление новой задачи по Enter
2 Отметка выполнения (чекбокс) и массовое переключение
3 Редактирование задачи по двойному клику (сохранение по Enter, отмена по Escape)
4 Удаление задачи (контекстное меню)
5 Контекстное меню при правом клике: редактировать, отметить/удалить отметку, удалить
6 Фильтры: Все / Активные / Завершённые (расположены в шапке)
7 Счётчик оставшихся задач
8 Очистка всех выполненных задач
9 Сохранение списка задач в localStorage
10 Переключение между светлой и тёмной темой
11 Адаптивная вёрстка (центрирование)
